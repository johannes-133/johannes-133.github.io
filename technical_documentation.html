Box appearance

There are different methods to taylor HTML elements to blend with page concept. Let's first look at these key aspects:

  • Padding
  • Margin
  • Border

The property padding allows you to set the spacing between any elements content and its edge. Here's how the padding property can be defined in the class .set-padding :

<style> .set-padding { padding: 10px; background-color: #FF7F00; } </style> <main> <div class="set-padding"> This element uses padding.</div> </main>
The code above would translate as follows when display in your web browser:
This element uses 20px padding.
You can use both relative and absolute length units to define padding:
  • mm (in millimeters, abs. )
  • px (in pixels) abs.
  • % (in percentage of parent element dim., rel. )
  • em (in font size of the element, rel. )
Try using the property padding for your self in your element.

Background customisation

HTML offers great ways to customise an elements background. The CSS property to be used in this context is the background property. You probably already know the background-color property. Let's look at these more sophisticated methods to style our element's background:

  • the linear-gradient() function
  • the url() function

With the linear-gradient function you can set a color transition for the background element. The function has at least three arguments. Depending on the number of colors to be used in the transistion, further arguments can be added:

  1. background: linear-gradient (gradient_direction, color 1, color 2, color 3, ...));
  2. background: linear-gradient(45deg, deepskyblue, lightgray, darkviolet);
The argument gradient_direction specifies the direction of the color transistion. The value 0deg results in a transition from bottom to top and the value 90deg in a transition from left to right. Further increasing the gradient value will result in a clockwise rotation of the transition direction. Using the second list to style an element will result in the following color display:

Bezier curves

You may have already learned about the @keyframes rule for animations. It is used to describe a particular animation, such as the animation sample-animation in the following example:

@keyframes: sample-animation{ 0%{ background-color: blue; } 100%{ background-color: black; } }
You have also learned about linking an animation to a class or id an the relevant attributes in this context:
animation-name: example-animation; animation-duration: 5s; animation-iteration-count: 10; animation-timing-function: ease-out;

We will now show how to use a bezier curve to customize the movement HTML objects. Let's say that we want an object ot bounce on a suface. In this case we want the speed to be highest, when the ball is about to tough the survace. When when the ball has maximum height, the speed tends to zero before inverting. The following CSS code represents this behavior for the class demo-bounce :

.demo-bounce{ animation-name: bounce; animation-duration: 3s; animation-iteration-count: infinite; animation-fill-mode: both; animation-timing-function: cubic-bezier(0,0,0.58,1); height: 25px; width: 25px; border-radius: 50%; }

Media queries

The last tip is on media queries. This tool can be very helpful to present to the user a different website layout according to his device. We all know the frustration of reading a regular website on our smartphone. We need to zoom in a lot to be able to read the content. Media queries @media in your css stylesheet can be one way of adressing this issue. @media is used in combination with a condition, for example the screen width.

@media (max-width: 800px){ #main-doc{ width: 100%; transition: margin-left .5s; margin: 0 0 0 20%; } }
The code above shows, how @media is applied to the id #main-doc , giving it a list of attribute values in case the viewport width is smaller than 800 pixels. Many older smarphones, such as the iPhone 4 have a screen width smaller than 800 pixels. Use media queries, when designing a website intended for both smartphone and regular pc users :).
